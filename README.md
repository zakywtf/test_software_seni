# step to use

1. npm install --save
2. you can run project using <br/>
    a. npm start <br/>
    b. npm run debug <br/>
    c. npm run debug >> /dev/null <br/> <br/>

A. get articles<br/>
    use url http://localhost:2500/api/v1/articles to get article. using raw json in your payload. example like this : <br/>
